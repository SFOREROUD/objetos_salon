Objetivos del Sistema

Este sistema implementa un modelo de Deep Learning (SSD MobileNetV2) entrenado para detectar y contabilizar automáticamente los objetos presentes en el salón de cómputo. El modelo se ejecuta completamente de forma local utilizando TensorFlow Lite.
